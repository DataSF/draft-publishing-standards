The following column names should be used only if they adhere to the definitions in this guide:

- year
- half_year
- quarter
- month
- week
- date
- datetime
- time
- fiscal_year
- fiscal_half_year
- fiscal_quarter
- latitude
- longitude
- x_coord
- y_coord
- zip_code
- supervisor_district
- unit
- unit_suffix
